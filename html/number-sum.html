Math Fact

Ever wonder why the value of the result is only one number? The answer is simple: we add the values that are in an array. It's that simple!

For example, let's assume that the array contains the numbers 2, 10, 4, 5, 1, and 2. 2 + 10 = 12; 12 + 4 = 16; 16 + 5 = 21; 21 + 2 = 22; and 22 + 2 = 24. So the value for the result is 24. Easy and free!
